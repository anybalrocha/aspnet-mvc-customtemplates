# ASP.NET MVC - Customizando DisplayFor e EditFor

Projeto destinado aos desenvolvedores ASP.NET MVC, com o objetivo de apresentar os recursos de customização do DisplayFor e do EditFor. Onde você desenvolve seu componente e consegue aplicar este nas suas views, atreladas ao tipo da sua propriedade no seu Model ou ViewModel desta view; aproveitando o código e centralizando a lógica de apresentação deste tipo de dado. 

Esta abordagem concede ao desenvolvedor maior produtividade no desenvolvimento de sistemas e facilita a manutenção do código relacionado a este ponto no seu projeto, além de conceder maior liberdade quanto a apresentação e comportamento do tipo de dado nas suas views.
